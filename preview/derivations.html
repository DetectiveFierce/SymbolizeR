Step-by-Step Derivations

library(SymbolizeR)

SymbolizeR’s derivation functions show every step of the calculation—invaluable for learning, teaching, and verification.

derive.E() — Expectation Derivation

derive.E(a * X + b)
#> Derivation of: E(a * X + b) 
#> ==================================================
#> 
#> Step 1: Starting expression
#>   Rule: Input
#>   => E(a * X + b)
#> 
#> Step 2: Apply expectation rules
#>   Rule: Linearity: E[aX + b] = aE[X] + b, E[X + Y] = E[X] + E[Y]
#>   => a * E(X) + b
#> 
#> Step 3: Substitute known distributions
#>   Rule: If X ~ Distribution, replace E[X] with known moment
#>   => a * E(X) + b
#> 
#> Step 4: Final result
#>   Rule: Done
#>   => a * E(X) + b
#> 
#> ==================================================
#> Final Answer: a * E(X) + b

derive.Var() — Variance Derivation

Shows complete variance calculation using Var(X) = E[X2] − E[X]2:

derive.Var(X)
#> Derivation of: Var(X) 
#> ==================================================
#> 
#> Step 1: Starting expression
#>   Rule: Input
#>   => Var(X)
#> 
#> Step 2: Apply variance definition
#>   Rule: Var[X] = E[X²] - E[X]²
#>   => E[X^2] - E[X]²
#> 
#> Step 3: Compute E[X²]
#>   Rule: Apply expectation rules to X²
#>   => E(X^2) - E[X]²
#> 
#> Step 4: Compute E[X]²
#>   Rule: Apply expectation rules to X, then square
#>   => E(X^2) - E(X)^2
#> 
#> Step 5: Combine terms
#>   Rule: E[X²] - E[X]²
#>   => E(X^2) - E(X)^2
#> 
#> Step 6: Final result
#>   Rule: Done
#>   => E(X^2) - E(X)^2
#> 
#> ==================================================
#> Final Answer: E(X^2) - E(X)^2

Linear Transformations

See polynomial expansion for Var(aX + b):

derive.Var(a * X + b)
#> Derivation of: Var(a * X + b) 
#> ==================================================
#> 
#> Step 1: Starting expression
#>   Rule: Input
#>   => Var(a * X + b)
#> 
#> Step 2: Apply variance definition
#>   Rule: Var[X] = E[X²] - E[X]²
#>   => E[(a * X + b)^2] - E[a * X + b]²
#> 
#> Step 3: Expand polynomial
#>   Rule: (a + b)² = a² + 2ab + b²
#>   => E[(a * X)^2 + 2 * (a * X * b) + b^2] - E[a * X + b]²
#> 
#> Step 4: Compute E[X²]
#>   Rule: Apply expectation rules to X²
#>   => E((a * X)^2) + 2 * (a * E(X) * b) + b^2 - E[a * X + b]²
#> 
#> Step 5: Compute E[X]²
#>   Rule: Apply expectation rules to X, then square
#>   => E((a * X)^2) + 2 * (a * E(X) * b) + b^2 - (a * E(X) + b)^2
#> 
#> Step 6: Combine terms
#>   Rule: E[X²] - E[X]²
#>   => E((a * X)^2) + 2 * (a * E(X) * b) + b^2 - (a * E(X) + b)^2
#> 
#> Step 7: Final result
#>   Rule: Done
#>   => E((a * X)^2) + 2 * (a * E(X) * b) + b^2 - (a * E(X) + b)^2
#> 
#> ==================================================
#> Final Answer: E((a * X)^2) + 2 * (a * E(X) * b) + b^2 - (a * E(X) + b)^2

With Defined Distribution

clear.definitions()
define(X ~ Normal(mu, sigma))

derive.Var(X)
#> Derivation of: Var(X) 
#> ==================================================
#> 
#> Step 1: Starting expression
#>   Rule: Input
#>   => Var(X)
#> 
#> Step 2: Apply variance definition
#>   Rule: Var[X] = E[X²] - E[X]²
#>   => E[X^2] - E[X]²
#> 
#> Step 3: Compute E[X²]
#>   Rule: Apply expectation rules to X²
#>   => sigma^2 + mu^2 - E[X]²
#> 
#> Step 4: Compute E[X]²
#>   Rule: Apply expectation rules to X, then square
#>   => sigma^2 + mu^2 - mu^2
#> 
#> Step 5: Combine terms
#>   Rule: E[X²] - E[X]²
#>   => sigma^2 + mu^2 - mu^2
#> 
#> Step 6: Simplify
#>   Rule: Algebraic simplification
#>   => sigma^2
#> 
#> Step 7: Final result
#>   Rule: Done
#>   => sigma^2
#> 
#> ==================================================
#> Final Answer: sigma^2
clear.definitions()

derive.Cov() — Covariance Derivation

Shows covariance using Cov(X, Y) = E[XY] − E[X]E[Y]:

derive.Cov(X, Y)
#> Derivation of: Cov(X, Y) 
#> ==================================================
#> 
#> Step 1: Starting expression
#>   Rule: Input
#>   => Cov(X, Y)
#> 
#> Step 2: Apply covariance definition
#>   Rule: Cov[X, Y] = E[XY] - E[X]E[Y]
#>   => E[X * Y] - E[X] * E[Y]
#> 
#> Step 3: Compute E[XY]
#>   Rule: Apply expectation rules to product
#>   => E(X * Y) - E[X] * E[Y]
#> 
#> Step 4: Compute E[X] * E[Y]
#>   Rule: Apply expectation rules to each variable
#>   => E(X * Y) - E(X) * E(Y)
#> 
#> Step 5: Combine terms
#>   Rule: E[XY] - E[X]E[Y]
#>   => E(X * Y) - E(X) * E(Y)
#> 
#> Step 6: Final result
#>   Rule: Done
#>   => E(X * Y) - E(X) * E(Y)
#> 
#> ==================================================
#> Final Answer: E(X * Y) - E(X) * E(Y)

Understanding the Output

Each derivation returns a derivation object:

Component | Description |

derive.Cov(X, X + Y) ```

It correctly identifies that Cov(X, X) becomes Var(X).

Why Use This?

  1. Debugging Math: If your hand-derived result doesn’t match your simulation, use this to find the line where you made a mistake.
  2. Learning: It helps build intuition for why certain terms cancel out.
  3. Documentation: You can copy-paste these derivations directly into your homework or reports.
